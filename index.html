About scrollMe.js

scrollMe.js is a small jquery snippet that allows you to easily set internal smooth-scroll links. To keep this super simple to add and update, all options are declared within the anchor tag, no need to mess with any other javascript other than loading up the script.


How to Use

  1. Create an anchor tag

    2. <a href="#">Go ahead, try me, give me a click </a>

  2. Add the "scrollme" class

    3. <a class="scrollme" href="#">Go ahead, try me, give me a click </a>

  3. Set the speed to scroll to your destination in milliseconds with the "data-speed" attribute

    4. <a class="scrollme" data-scroll-speed="900" href="#"> Go ahead, try me, give me a click </a>

  4. Choose the scroll method and distance:

There are 4 types of scroll methods you can choose from

  1. Scroll x number of pixels from the top of the page (absolute)
  2. Scroll x number of pixels from the link being clicked (relative)
  3. Scroll directly to a class or id tag (specific)
  4. Scroll to a class or id with an additional distance involved (abSpecific)

Each link contains a "data-scroll-type" attribute which determines the method of scrolling used, and (all but one scroll-type) a "data-scroll-y" attribute which sets the distance travelled vertically on the page.

data-type="absolute"

To scroll a distance from the top of the page, set the "data-type" attribute to "absolute". Give the "data-scroll-y" attribute a value of pixels to travel from the top of the page.

<a class="scrollme" data-scroll-speed="900" data-scroll-type="absolute" data-scroll-y="1400" href="#"> Go ahead, try me, give me a click </a>

data-type="relative"

To scroll a distance relative to the link being clicked, set the "data-type" attribute to "relative". Give the "data-scroll-y" attribute a value of pixels you want to travel from the link being clicked. Note: you can use positive or negative values.

<a class="scrollme" data-scroll-speed="900" data-scroll-type="relative" data-scroll-y="400" href="#"> Go ahead, try me, give me a click </a>

data-type="specific"

To scroll directly to a class or id, set the "data-type" attribute to "specific". Additionally, set the "data-gotothis" attribute to the class or ID you wish to scroll to. You do not need a "data-scroll-y" attribute for this method.
Note: if you have several classes on one page, the link will go to the first class it comes across. In the interest of safety, you may want to use IDs only, or and ID in addition to any class based tags you wish to scroll to.

<a class="scrollme" data-scroll-speed="900" data-scroll-type="specific" data-gotothis="#mytag" href="#"> Go ahead, try me, give me a click </a>

data-type="abSpecific"

The "abSpecific" option is a hybrid between absolute scrolling and specific scrolling. Say you have a class or ID you wish to scroll to, but you want the user to land 100 pixels before the beginning of that section, or 100 pixels after. By setting the "data-type" to "abSpecific" you get this control. Set the "data-gotothis" just as you would for the "specific" option, then set the "data-scroll-y" to the additional distance you want traveled. Note: you can use positive or negative values.

Additional Options

Relative scroll option:

The default scrollme "goto" data attribute will take you x number of pixels (that you declare in data-scroll-xPixels) from the top of the page. If you would like to set the destination x number of pixels relative to the link being clicked, use the "data-rel-goto" option.

For example:

<a class="scrollme" data-speed="2000" data-rel-goto="200" href="#">Go ahead, try me, give me a click </a>
