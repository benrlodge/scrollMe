scrollMe.js

About scrollMe.js

scrollMe.js is a small jquery plugin that allows you to easily set smooth-scroll internal links. scrollMe.js was developed by Ben Lodge.


Download the source on Github

How to Use

  1. Create an anchor tag

    2. <a href="#"> Go ahead, try me, give me a click </a>

  2. Add an id to represent the link

    3. <a id="mylink" href="#">Go ahead, give me a click </a>

  3. Set the scroll method and distance. There are 2 types of scroll methods you can choose from:
  1. Scroll x number of pixels from the top of the page (absolute)
  2. Scroll directly to a class or id tag (specific)

$("#introdemo").scrollMe({
type: "absolute", //absolute or specific speed: 1900, //speed in milliseconds scrollto: 1297 //pixels to travel... scrollto: '#gohere' //or id to travel to })

Last, load up the scripts

<script src="jquery.js"> </script>
<script src="fixedScrollNav.js"> </script>

Demos

I am a specific link. I travel up to the about section.

$("#specific-demo").scrollMe({
type: "specific", scrollto: "#about" })

I am an absolute link. I travel 800px from the top of the page.

$("#absolute-demo").scrollMe({
type: "absolute", scrollto: 800 })
